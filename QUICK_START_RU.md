# 🚀 Быстрый Старт - SSH-АРЕНА

## 🎮 Запуск Игры

### Шаг 1: Запустите Room Manager
```bash
./room-manager.exe
```

### Шаг 2: Запустите Gateway
```bash
./gateway.exe
```

### Шаг 3: Подключитесь через SSH
```bash
ssh localhost -p 2222
```

## 🎨 Новый Интерфейс

Теперь игра имеет крутой красный SSH-стиль с:
- ✨ Анимированным баннером "SSH-АРЕНА"
- 🎯 Красивыми меню с градиентами
- ⚡ Индикаторами загрузки
- 🔥 Мигающими эффектами
- 🇷🇺 Полностью на русском языке

## 📋 Управление

- `W` - Вперед
- `S` - Назад
- `A` - Поворот влево
- `D` - Поворот вправо
- `SPACE` - Стрелять
- `Q` - Выход
