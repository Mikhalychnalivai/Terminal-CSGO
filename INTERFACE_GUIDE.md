# 🔴 SSH-АРЕНА - Красный Интерфейс

## 🎨 Новый Дизайн

Игра теперь имеет крутой красный SSH-стиль с градиентами и анимацией!

### ✨ Особенности интерфейса:

#### 1. **Стартовый Баннер**
- Анимированный ASCII-арт логотип SSH-АРЕНА
- Красный градиент от яркого к темному
- Плавное появление строк

#### 2. **Меню Выбора**
- Красивые рамки с градиентом
- Иконки для каждого пункта меню:
  - ⚔ СОЗДАТЬ НОВУЮ АРЕНУ
  - 🚪 ПРИСОЕДИНИТЬСЯ К АРЕНЕ
  - 🗺 ВЫБЕРИТЕ КАРТУ
- Цветовое выделение опций
- Все надписи на русском языке

#### 3. **Экран Подключения**
- Анимированный индикатор загрузки (⠋⠙⠹⠸⠼⠴⠦⠧⠇⠏)
- Сообщения о статусе подключения
- Красивые сообщения об ошибках

#### 4. **Экран Успешного Подключения**
- Зеленая галочка ✓ для успеха
- Отображение информации:
  - Позывной (никнейм игрока)
  - ID Арены (ID комнаты)
- Мигающий эффект "ГОТОВ К БОЮ"
- Призыв к действию: ⚔ НАЖМИТЕ ENTER ДЛЯ ВХОДА ⚔

#### 5. **Выбор Карты**
- Варианты при создании комнаты:
  - `[1]` corridor5 — коридор, 5 комнат (по умолчанию)
  - `[2]` свой `map_id` (как у API `room-manager`)
  - DOOM.WAD — только если WAD добавлен в образ / `WAD_PATH` настроен у `room`

### 🎮 Как Запустить

1. Запустите сервер room-manager:
```bash
./room-manager.exe
```

2. Запустите gateway:
```bash
./gateway.exe
```

3. Подключитесь через SSH:
```bash
ssh localhost -p 2222
```

### 🌈 Цветовая Схема

- **Основной цвет**: Красный (#FF0000)
- **Градиенты**: От яркого красного (255,0,0) до темного (150,0,0)
- **Акценты**: 
  - Зеленый для успеха (100,255,100)
  - Желтый для информации (255,255,100)
  - Серый для подсказок (150,150,150)

### 🎯 Анимации

- **Появление баннера**: 40ms задержка между строками
- **Меню**: 30ms задержка между элементами
- **Загрузка**: 80ms между кадрами анимации
- **Мигание**: 300ms между состояниями

### 📝 Примечания

- Интерфейс оптимизирован для терминалов с поддержкой True Color (24-bit)
- Рекомендуемый размер терминала: минимум 80x24
- Курсор автоматически скрывается во время анимаций
