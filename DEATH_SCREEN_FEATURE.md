# 💀 Экран Смерти с Красным ASCII Фоном

## 🎨 Новая Функция

Добавлен красивый плавный красный ASCII фон для экрана смерти с анимацией и эффектами!

## ✨ Особенности

### 1. Градиентный Красный Фон
- Плавный градиент от темно-красного к ярко-красному
- ASCII паттерны: `░ ▒ ▓ █ ▓ ▒ ░`
- Вариация цвета по горизонтали для объема
- Анимированное появление (15ms между строками)

### 2. Минималистичный Дизайн
- Чистый фон без отвлекающих элементов
- Фокус на главном сообщении "ВЫ ПОГИБЛИ!"
- Белый текст с тенью на красном фоне
- Плавное появление построчно (40ms)

### 3. Мигающее Сообщение
- "ВЫ ПОГИБЛИ!" мигает 5 раз
- Переключение между белым (255,255,255) и темно-красным (150,0,0)
- Интервал мигания: 250ms

### 4. Информация об Убийце
- Отображается имя убийцы (если есть)
- Светло-красный цвет текста (255,200,200)

## 🎯 Технические Детали

### Градиент Фона
```go
// Вертикальный градиент
if progress < 0.5 {
    r = 100 + (155 * progress * 2)  // Темный → Яркий
} else {
    r = 255 - (155 * (progress - 0.5) * 2)  // Яркий → Темный
}
```

### ASCII Паттерны
```
░ - Легкая тень (25%)
▒ - Средняя тень (50%)
▓ - Плотная тень (75%)
█ - Полный блок (100%)
```

### Цветовая Схема
- **Фон**: RGB(80-255, 0, 0) - градиент
- **"ВЫ ПОГИБЛИ!"**: RGB(255, 255, 255) → RGB(150, 0, 0) - мигание
- **Тень**: RGB(80, 0, 0) - темно-красный
- **Текст**: RGB(255, 200, 200) - светло-красный

## 🎬 Анимация

1. **Фон** (360ms):
   - 24 строки × 15ms = 360ms
   - Появляется сверху вниз

2. **Текст** (640ms):
   - ~16 строк × 40ms = 640ms
   - Появляется построчно с тенью

3. **Мигание** (2500ms):
   - 5 циклов × (250ms + 250ms) = 2500ms
   - Привлекает внимание к сообщению

**Общее время**: ~3.5 секунды

## 🎮 Использование

Экран смерти автоматически появляется когда:
- Игрок погибает в бою
- Здоровье достигает 0
- Получен фатальный урон

После нажатия ENTER игрок возрождается в случайной точке спавна.

## 🔧 Настройки

Можно изменить в коде:
- `termWidth` и `termHeight` - размер экрана
- Скорость анимации фона (15ms)
- Скорость появления текста (40ms)
- Скорость мигания (300ms)
- Количество циклов мигания (3)

## 💡 Советы

- Используйте терминал с поддержкой True Color (24-bit)
- Рекомендуемый размер: минимум 80x24
- Лучше всего смотрится в полноэкранном режиме
