# ✅ Исправление: Видимый Ввод

## Проблема
При вводе текста (имя, ID арены, выбор карты) символы не отображались на экране.

## Решение
Изменены ANSI escape-коды для правильного отображения ввода:

### Было:
```go
io.WriteString(s, "\x1b[38;2;255;120;120m> \x1b[38;2;255;255;255m\x1b[?25h")
```

### Стало:
```go
io.WriteString(s, "\x1b[38;2;255;120;120m> \x1b[0m\x1b[?25h")
```

## Изменения

Использование `\x1b[0m` вместо `\x1b[38;2;255;255;255m` для сброса всех атрибутов текста, что позволяет терминалу использовать стандартный цвет для ввода.

### Исправлено в:
1. Ввод позывного (никнейма)
2. Выбор режима [1/2]
3. Ввод ID арены
4. Выбор карты

## Результат
✅ Теперь весь вводимый текст виден пользователю
✅ Курсор отображается корректно
✅ Цвет приглашения (>) остается красным
✅ Вводимый текст отображается стандартным цветом терминала

## Технические Детали

ANSI коды:
- `\x1b[38;2;R;G;Bm` - установить цвет текста RGB
- `\x1b[0m` - сбросить все атрибуты (цвет, стиль)
- `\x1b[?25h` - показать курсор
- `\x1b[?25l` - скрыть курсор

## Тестирование

Проверьте что:
1. При вводе имени символы видны
2. При выборе режима цифры видны
3. При вводе ID арены текст виден
4. При выборе карты ввод отображается
