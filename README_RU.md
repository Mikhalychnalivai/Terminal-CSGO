# 🔴 SSH-АРЕНА

Многопользовательская боевая арена в стиле DOOM, доступная через SSH с красивым красным интерфейсом!

## 🎮 Особенности

- 🌐 Подключение через SSH (порт 2222)
- 🎨 Красный градиентный интерфейс с анимациями
- 🗺️ Карта по умолчанию — JSON **corridor5** (в образе); опционально **DOOM.WAD**
- ⚔️ Многопользовательский режим
- 🔥 Полностью на русском языке
- 🎯 3D рендеринг в ASCII
- 📊 Система статистики игроков (убийства, смерти, точность)

## 🚀 Быстрый Старт

### 1. Запустите сервер управления комнатами
```bash
./room-manager.exe
```

### 2. Запустите шлюз
```bash
./gateway.exe
```

### 3. Подключитесь через SSH
```bash
ssh localhost -p 2222
```

## 🎯 Управление

- `W` - Движение вперед
- `S` - Движение назад
- `A` - Поворот влево
- `D` - Поворот вправо
- `SPACE` - Выстрел
- `R` - Перезарядка выбранного оружия: пистолет — обойма до 10 (запас бесконечен); автомат — из запаса, если в магазине меньше 30
- `1` / `2` - Пистолет / автомат
- `Q` - Выход из игры
- `TAB` - Просмотр статистики игрока
- `B` - Магазин (покупка патронов и брони)
- `ESC` - Закрыть открытое меню

## 🗺️ Карты

- **corridor5.json** — карта по умолчанию в Docker-образе
- **DOOM.WAD** — при подключении WAD в образе (см. `map/README_DOOM_WAD.txt`, при наличии)

## 📊 Система Статистики

Игра автоматически собирает статистику всех игроков:

### Статистика Игрока
- **Убийства** — общее количество убийств
- **Смерти** — общее количество смертей
- **K/D Ratio** — соотношение убийств к смертям
- **Точность** — процент попаданий от всех выстрелов
- **Выстрелов/Попаданий** — детальная статистика стрельбы
- **Pistol/Rifle Kills** — убийства по типам оружия
- **Время в игре** — общее время сессий

### Просмотр Статистики
Нажмите `TAB` во время игры для просмотра вашей статистики.

### API Статистики
Statistics API доступен на `http://room-manager:8081/stats/`:

| Endpoint | Описание |
|----------|----------|
| `GET /stats/player/{id}` | Статистика игрока |
| `GET /stats/players/top?limit=10` | Топ игроков |
| `GET /stats/weapons` | Статистика оружия |
| `GET /stats/maps` | Статистика карт |
| `GET /stats/export/players` | Экспорт в CSV |
| `GET /stats/export/events` | Экспорт событий (JSON lines) |

### Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `STATS_ENABLED` | Включить статистику | `true` |
| `STATS_DB_PATH` | Путь к базе данных | `/data/stats.db` |
| `STATS_ENDPOINT` | Endpoint для room-контейнеров | `http://room-manager:8081/stats/events` |

## 🌈 Цветовая Схема

- 🔴 Красный градиент: RGB(255,0,0) → RGB(150,0,0)
- ✅ Успех: RGB(100,255,100)
- 💛 Информация: RGB(255,255,100)
- ⚪ Текст: RGB(255,255,255)
- ⚫ Подсказки: RGB(150,150,150)

## 🎬 Анимации

- Баннер: 40ms между строками
- Меню: 30ms между элементами
- Загрузка: 80ms между кадрами
- Мигание: 300ms между состояниями

## 🐛 Решение Проблем

### Не могу подключиться
- Убедитесь что room-manager и gateway запущены
- Проверьте что порт 2222 свободен

### Плохое качество графики
- Используйте терминал с поддержкой True Color
- Рекомендуется: Windows Terminal, iTerm2, Alacritty

### Лаги или задержки
- Проверьте сетевое подключение
- Убедитесь что сервер запущен локально
